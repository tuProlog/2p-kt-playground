# 2P-Kt Playground
2P-Kt Playground Web project
## Install
```bash
  npm install
```
## Usage


```bash
  "scripts": {
    "start": "...",
    "build": "...",
    "test": "..."
  },
```
### npm start
* start Webpack Dev server using unminified sourcers('js')
### npm run build
* run test
* build and create production ready artifacts in `prodDist` folder.
### npm test
* runs mocha test
